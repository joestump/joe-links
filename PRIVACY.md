# Privacy Policy

**Last updated:** April 4, 2026

## Overview

joe-links is a self-hosted "go links" service. It is designed to run entirely on infrastructure you own and control. There is no hosted version, no telemetry, and no data collection of any kind by the authors of this software.

## Data Collection

We do not collect, store, transmit, or process any user data. Period.

- No analytics or telemetry
- No tracking pixels or cookies sent to third parties
- No crash reports sent externally
- No personal information gathered

## Data Storage

All data created by joe-links (links, user accounts, sessions, API tokens) is stored exclusively in the database you configure on your own infrastructure. The authors of this software have no access to your data.

## Browser Extension

The joe-links browser extension communicates solely with the joe-links server you configure in its settings. It stores your server URL, API key, and cached keyword list locally in your browser using `chrome.storage.local`. No data is transmitted to the extension authors or any third party.

## Data Selling

We do not sell, rent, trade, or otherwise share any user data. We do not have access to any user data in the first place.

## Self-Hosting

joe-links is intended exclusively for self-hosting. You are solely responsible for the security, availability, and privacy practices of your own deployment. We strongly recommend running joe-links behind HTTPS and restricting access to trusted networks.
